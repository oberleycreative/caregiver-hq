Easy to use and configure, multilevel offcanvas mobile navigation menu plugin.

== Description ==

= GT Push Menu plugin was designed to bring the best mobile experience to your visitors. =

Unlike other mobile menu plugins it uses easy to understand visual layout and nice animation.
With this plugin your visitors won't get lost in multiple levels of the menu.

= GT Push Menu Pro =

**GT Push Menu Pro** is a significant upgrade to **GT Push Menu Lite** that allows you to use custom icon fonts, use image in the menu bar and allows you to enable additional menu panel which slides from the right. Additional menu panel is a standard WordPress sidebar which gives you the opportunity to use standard widgets in it.

You can learn more about *GT Push Menu Pro* and see the difference in versions here:
[Plugin Versions Comparison](http://griffinthemes.com/ "Plugin Versions Comparison")

== Installation ==

= 1. Install through WordPress Add menu =
1. In your Admin, go to menu Plugins > Add
1. Search for 'gt-push-menu'
1. Click to install
1. Activate the plugin

= 2. Install through WordPress Upload menu =
1. Download the plugin (.zip file) on the right column of this page
1. In your Admin, go to menu Plugins > Add
1. Select the tab "Upload"
1. Upload the .zip file you just downloaded
1. Activate the plugin

= 3. Install through FTP =
1. Download the plugin (.zip file) on the right column of this page
1. Unzip gt-push-menu.zip to temp folder
1. Upload 'gt-push-menu' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. A new menu 'GT Push Menu Lite' will appear in your Admin

Go to *Appearance* -> *Menus* and configure the plugin in the "GT Push Menu" section on the left.
